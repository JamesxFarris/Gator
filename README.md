# Gator

A mobile life improvement pet app built with React Native and TypeScript.

## About

Gator is your digital companion designed to help you build better habits, stay motivated, and improve your daily life through an engaging pet experience.

## Tech Stack

- React Native
- TypeScript

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- React Native CLI
- Xcode (for iOS development)
- Android Studio (for Android development)

### Installation

```bash
# Install dependencies
npm install

# iOS only: Install CocoaPods
cd ios && pod install && cd ..

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Project Structure

```
src/
  components/    # Reusable UI components
  screens/       # App screens
  navigation/    # Navigation configuration
  services/      # API and business logic
  hooks/         # Custom React hooks
  utils/         # Utility functions
  assets/        # Images, fonts, etc.
  types/         # TypeScript type definitions
```
